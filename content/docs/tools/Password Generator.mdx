# IP-Bereichs-Rechner

Mit diesem Tool können Sie den IP-Adressbereich für eine gegebene IP-Adresse und CIDR-Notation berechnen.

import IpRangeCalculator from '@site/src/components/IpRangeCalculator';

<IpRangeCalculator />

## Wie es funktioniert

1. Geben Sie eine IP-Adresse ein (z.B. 192.168.1.1).
2. Geben Sie den CIDR-Wert ein (z.B. 24 für eine /24 Subnetzmaske).
3. Klicken Sie auf "Berechnen".
4. Das Tool zeigt Ihnen die Start- und End-IP des Bereichs an.

Dieses Tool ist nützlich für Netzwerkadministratoren und alle, die mit IP-Adressbereichen arbeiten müssen.
